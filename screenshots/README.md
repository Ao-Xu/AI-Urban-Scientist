# Screenshots

Place your screenshot files in this directory.

## Suggested filenames

| File | Description |
|------|-------------|
| `01-home.png` | Main page — input form before submission |
| `02-running.png` | Pipeline running — progress log and stage indicators |
| `03-idea-done.png` | After Stage 1 — Idea download button active |
| `04-download.png` | All stages complete — all three download buttons active |
| `05-pdf-preview.png` | Example of a generated PDF |

## How to reference in README

Once you add a screenshot, update `README.md` by replacing the placeholder comment with:

```markdown
![Home page](screenshots/01-home.png)
![Running](screenshots/02-running.png)
![Download](screenshots/04-download.png)
```
